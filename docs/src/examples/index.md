## Running the Examples

The full example scripts (with visualization and animation) are available in the `examples/` directory of the repository:

```julia
# From the root of the repository
include("examples/rotating_circle.jl")
include("examples/zalesak_disk.jl")
include("examples/colliding_balls.jl")
```

The scripts require `GLMakie` and `FFMPEG` for visualization and video generation.
