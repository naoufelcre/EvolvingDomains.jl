Geometric Quantities

User-Facing Functions

EvolvingDomains.interface_curvature — Function
interface_curvature(eg::EvolvingDiscreteGeometry) -> CellField

Return curvature as a CellField defined on the background mesh. Can be evaluated on the interface Γ in variational forms.

Uses cached computation - only recomputes when geometry changes.

Example

cut_geo = current_cut(eg)
Γ = EmbeddedBoundary(cut_geo)
n_Γ = get_normal_vector(Γ)
κ_Γ = interface_curvature(eg)

dΓ = Measure(Γ, 2)
σ = 0.072  # Surface tension
∫( σ * κ_Γ * (n_Γ ⋅ v) )dΓ
source
EvolvingDomains.interface_normal — Function
interface_normal(eg::EvolvingDiscreteGeometry) -> CellField

Return the normal vector on the embedded boundary Γ. This is a thin wrapper around GridapEmbedded's get_normal_vector.

Example

cut_geo = current_cut(eg)
Γ = EmbeddedBoundary(cut_geo)
n_Γ = interface_normal(eg)  # Equivalent to get_normal_vector(Γ)
source
EvolvingDomains.curvature_at_band — Function
curvature_at_band(eg::EvolvingDiscreteGeometry; n_layers=2) -> (Vector{Float64}, Set{Int})

Compute curvature only in the narrow band around the interface.

Returns

  • κ_full: Full-length vector with curvature values (zeros outside band)
  • band_nodes: Set of node indices where curvature was computed

Example

κ, band = curvature_at_band(eg)
# κ is ready to create FE function: FEFunction(V, κ)

EvolvingDomains.expand_to_band — Function
expand_to_band(cut_cells, n_layers, partition) -> Vector{Int}

Expand a set of cell indices by n_layers to form a narrow band. Uses 4-neighbor connectivity for Cartesian grids.

Arguments

  • cut_cells: Initial cell indices (typically CUT cells)
  • n_layers: Number of layers to expand (default: 2 for curvature stencil)
  • partition: Grid cell counts (nxcells, nycells)
source
EvolvingDomains.cells_to_nodes — Function
cells_to_nodes(cells, partition) -> Set{Int}

Get all node indices belonging to a set of cells. Assumes bilinear (4-node) quad cells on Cartesian grid.

Arguments

  • cells: Cell indices
  • partition: Grid cell counts (nxcells, nycells)

Usage

using EvolvingDomains, Gridap, GridapEmbedded

# Setup
eg = EvolvingDiscreteGeometry(evolver, model)

# Get curvature as CellField (cached, narrow-band computation)
κ_Γ = interface_curvature(eg)

# Get normal from GridapEmbedded
n_Γ = interface_normal(eg)

# Surface tension in weak form
cut_geo = current_cut(eg)
Γ = EmbeddedBoundary(cut_geo)
dΓ = Measure(Γ, 2)
σ = 0.072

# Use in variational form
a(u, v) = ∫( σ * κ_Γ * (n_Γ ⋅ v) )dΓ

Narrow Band Details

Curvature is computed only in a narrow band around the interface:

ParameterDefaultDescription
n_layers2Layers of cells beyond CUT cells

Low-Level Access

# Get curvature values and band info
κ_values, band_nodes = curvature_at_band(eg; n_layers=2)

# κ_values: full-length Vector{Float64} (zeros outside band)
# band_nodes: Set{Int} of node indices where curvature was computed
